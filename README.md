# Portfolio E4 — BTS SIO SISR (Next.js + TypeScript + Tailwind)

## Installation
- npm install

## Lancement local
- npm run dev
Puis ouvrir http://localhost:3000

## Build
- npm run build
- npm run start

## Déploiement Vercel
- Importer le repo dans Vercel
- Build command: npm run build
- Le reste est automatique

## Où modifier les contenus
- Profil : src/data/profile.ts
- Compétences : src/data/competences.ts
- Réalisations + preuves : src/data/realisations.ts
- PDFs : public/preuves/*.pdf
